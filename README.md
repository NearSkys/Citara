# Referências Projeto

Repositório para versionamento do projeto de referências.

Como usar (local):

- Inicializar (já feito): `git init -b main`
- Adicionar remote: `git remote add origin <url-do-repositorio>`
- Enviar para remoto: `git push -u origin main`

Observações:
- Configure `git config user.name` e `git config user.email` globalmente se desejar commits com seu nome.
