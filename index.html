Citara: Gerenciador de Referências Acadêmicas

Ferramenta leve para organizar PDFs e DOIs, buscar artigos via Semantic Scholar e visualizar referências. Desenvolvido na Imersão Dev da Alura para concorrer à bolsa FIAP.

Abrir Citara

Contexto do Projeto

Citara foi desenvolvido como uma submissão para o programa de Imersão Dev da Alura, com o objetivo de concorrer a uma prestigiada bolsa de estudos na FIAP. O projeto representa um esforço dedicado em aprender e aplicar princípios modernos de desenvolvimento web dentro de um escopo focado.

Disclaimer

Este projeto é uma prova de conceito criada dentro do prazo limitado do programa de imersão. Como tal, não é uma aplicação com todos os recursos. O foco foi demonstrar as principais funcionalidades e princípios de design, em vez de construir uma ferramenta pronta para produção.
